CraftMyPDF Template Guide

Use this HTML as a visual reference when building your template in CraftMyPDF. Yellow highlighted items are {{variables}} you'll add in the editor.

  • {{customer_name}} - Customer's full name
  • {{address}} - Service address
  • {{completion_date}} - Date of completion
  • {{job_type}} - Type of service
  • {{#each windows}}...{{/each}} - Loop for each window comparison
  • {{before_photo_url}} / {{after_photo_url}} - Photo URLs
  • {{before_caption}} / {{after_caption}} - Photo captions
  • {{before_measurements}} - Window measurements
  • {{ai_summary}} - AI-generated summary text
  • {{#each checklist_items}}{{this}}{{/each}} - Work completed items

Quick Variable Reference for CraftMyPDF

Variable Type Description
{{customer_name}} Text Customer's full name
{{address}} Text Service address
{{job_type}} Text Window Replacement, Inspection, etc.
{{completion_date}} Text Formatted date string
{{windows}} Array Array of window objects (use with #each)
{{number}} Number Window number (inside #each windows)
{{before_photo_url}} Image URL Before photo (inside #each windows)
{{after_photo_url}} Image URL After photo (inside #each windows)
{{before_caption}} Text Before photo caption
{{after_caption}} Text After photo caption
{{before_measurements}} Text Window dimensions
{{ai_summary}} Text AI-generated summary paragraph
{{checklist_items}} Array Work completed items (use with #each)
